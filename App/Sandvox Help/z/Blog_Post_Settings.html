Blog Post Settings

To edit a blog's settings:

  1. Click the collection in the Site Outline to select it.
  2. Click the text of the blog post to select it.
  3. Open the Object Inspector.

Collection to index

Here you can drag the target to the main collection of your blog if it is not already done for you

Show at most

This is to set the number posts that will be seen on the main page of your collection. Useful for larger collections such as blogs where there may eventually be dozens of pages.

Layout

There are many different 'views' to choose from, also you can check the options that you would like for your posts.

- Shows the pages of your collection as a bulleted title list

- Shows your collection as a list of titles

- Shows your collection as a table

- Shows your articles in order with page titles

- Shows your articles in order with page titles and thumbnail images

Date Created / Date Modified
This will display the date created or modified
Permanent Links
Makes the date that is displayed a link
Comments
Adds Comments to your posts

Article Length

Slide the slider to the left or right to lengthen or shorten the length of the article that shows on the main page of your collection.

Hyperlink titles

Check this box if you would like the title of your pages to be links to the pages.

Collection Presets

Collections can be used in many ways. Sandvox provides some pre-configured collections for the most popular uses. To use one of these presets, you can either:

  • Create a new collection by clicking the "New" item in the toolbar and select the type of page you want.
  • Select an existing collection in the Site Outline and use the Page Inspector to change its settings.


If you use one of these presets, don't forget that you can still tweak the settings of the collection using the Page Inspector. The following presets are available:

Blog
Designed for use as a Blog, Podcast or Videocast. Please see the "Blogging with Sandvox" and "Podcasting and Videocasting with Sandvox" articles for more information.
Photo Album
A grid of photos in the collection. Please see "Working with Photo Albums in Sandvox" for more information.
