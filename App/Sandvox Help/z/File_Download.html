File Download

A File Download is used to host a file on your site. Visitors click on a link to the File Download, and depending on their web browser and the file type, it downloads immediately or opens for viewing in their browser.

Ways to create a File Download:

  • Click the "Pages" item in the toolbar and select "File Download."
  • Choose "Add Page → File Download" from the Site menu.
  • Drag a file from the Finder to the Site Outline. Note that this only works for files that Sandvox does not have an equivalent page type for. e.g. A movie file is not suitable, but a zip archive is.

If you create a File Download using either of the first two methods, you will need to set the actual file that is to be downloaded. This also applies if you wish to change the file later.

To set the file to be download:

  1. Select the File Download in the Site Outline.
  2. Open the Page Inspector.

  3. Click the "Choose…" button.

  4. Select a file.


External Files

Sometimes, you may wish to allow visitors to download a file that isn't actually part of your site. In such cases, a File Download is not suitable. Instead, you should use either a link or an External Link page.

  • Use an External Link page if you want the download to appear in an index, the Site Menu or the Site Outline.
  • Otherwise, use a link
