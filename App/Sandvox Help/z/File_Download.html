File Download

A File Download page is used to host a file on your site. Visitors click on a link to the File Download, and depending on their web browser and the file type, it downloads immediately or opens for viewing in their browser.

Ways to create a File Download:

  • Click the "Pages" item in the toolbar and select "File Download."
  • Choose "Add Page → File Download" from the Site menu.
  • Drag a file from the Finder to the Site Outline. (This only works for files for which Sandvox does not have an equivalent page type. For example, if you want to make a Video or text file available for download, dragging and dropping won't work, because it will create a video or text page. To create a download page for these kinds of files, use one of the first two methods above.)

If you create a File Download using either of the first two methods, you will need to set the actual file that is to be downloaded. This also applies if you wish to change the file later.

To set the file to be download:

  1. Select the File Download in the Site Outline.
  2. Open the Page Inspector. The top contains general page attributes. The bottom allows you to specify the file to be made available for download.

  3. Click the "Choose…" button.

  4. Select a file.


External Files

Sometimes, you may wish to allow visitors to download a file that isn't actually part of your site. In such cases, a File Download is not suitable. Instead, you should use either a link or an External Page.

  • Use an External Page if you want the download to appear in an index, the Site Menu or the Site Outline.
  • Otherwise, use a link.
