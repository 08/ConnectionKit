General Object Attributes

Introduction

To edit an object, you use the Inspector. This contains settings that are applicable to all types of objects and those that apply only to the current object's type.

For more information on the other, object type-specific attributes, please see the articles on each type of object.

The Wrap Tab

Object Placement

Here is where you choose where you want the object to be placed on the page.

  • Inline (moves with text)
  • Callout
  • Sidebar

Place the object in the Sidebar or within the page as a callout. Note that some page types do not allow callouts. If you choose it to be inline you can choose if you want the text to wrap around it, and how you want the text to wrap around it.

To change the position of an object you click and drag it.

Layout

Show Title
Check this box if you want your object to have the title above it
Show Captions
Check this box if you would like to have a caption (or short description) under the object.
Border
If the site's design supports borders, checking this box will display a border around the object.

The Metrics Tab

  • Size - Use this tab to adjust the size of the object in pixels or you can also select the object in the page or sidebar by clicking on it, and adjust the size by dragging the corners.
  • Original Size - Use this button to take the object back to it's original size.

The Link Tab

Some objects can also be used as links. You can link to:

Nothing
External URL
You need to enter in the URL of the page you want to link to
Email
You need to enter in an email address that you want the mail sent to
Page in your site
Drag the target to an page in the site outline
Full-size image
Sandvox will automatically make the full size image available when the object it clicked on

You can choose for many of these options to open the links in a new window

Make all links active

Check this box if you wan the links in your document to be active, you would then be able to hold the shift key while clicking on links to test them.

The Media / Object Tab

This is the tab that you use to set conditions that are specific to the object.

The file info for an image
The email address for a contact form
Your books in an Amazon list
