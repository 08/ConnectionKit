How do I make sidebar objects appear only on certain pages

Ways to add or remove a sidebar object on a page:

  • Select the page you want to affect in the site navigator
  • Open up page inspector, appearance tab
  • Check or uncheck the object in the sidebar list


Initially, an object will appear in the page it is created on and any child pages if it is a collection.
