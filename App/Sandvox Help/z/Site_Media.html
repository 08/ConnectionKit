Site Media

This article describes each of the settings available in the "Media" tab of the Site Inspector.

Banner

Some designs allow you to change their banner image. For such designs, drag an image here to set the banner. Leave the box empty to use the default image.

Logo

Drag an image here to set the site logo.

Do not show site title

Hides the site title from view. This is useful if the site's banner or logo contains the title of the site and displaying it twice would be unnecessary.

Note that the site title is still present, just invisible to the casual visitor. This is for accessibility reasons (so screen-readers can still read it), and for search engines to determine the title of the site.

Favicon

Drag an image here to set the site's Favicon.

Keep media originals with document

Sets which media files should be copied into the Sandvox document. Note that the setting you choose here will determine how much and what kind of work you will have to do if you later decide to move your site to another computer in order to edit it. Please see the "Media" article for more information.

Replace titles with graphical text

Toggles the usage of graphical text on compatible computers and allows you to adjust the size.
