External Page

This page describes the use of the External Page object, not the External Link Page. If you simply want to add a link within another type of page (such as a text page or photo page), please see the Link article.

An External Page Object displays a piece of another website on a page of your website.

Ways to add an External Page to a site:

  • Use the "Objects" items in the toolbar.
  • Choose "External Page" from the Insert menu.
  • Enter the URL into the Object Inspector.

Settings

You can change the URL of the page in Object Inspector. Also, change the size in the Metric Inspector.
