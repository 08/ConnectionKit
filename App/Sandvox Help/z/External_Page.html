External Page

This page describes the use of the External Page page or pagelet. If you simply want to add a link within another type of page (such as a text page or photo page), please see the Link article.

You can add an External Link (a link to a web site other than your own) on an External Link page or pagelet. An External Link page appears in the Site Outline like any other page, but acts only as a "placeholder" for the external webpage. You can then link to the External Link in the Site Outline.

By default, if you have it set in your Preferences, when you create a new External Page, Sandvox will read and insert the URL from the current window of your browser (for Safari, Opera, Camino, Firefox, and OmniWeb).

Ways to add an External Link to a site:

  • Use the "Pages" or "Pagelets" items in the toolbar.
  • Choose "Add Page" or "Add Pagelet" from the Site menu.
  • Highlight and drag the URL of the site you wish to incorporate from your web browser to the Site Outline.

Settings

To edit an External Page:

  1. Select the External Page in the Site Outline.
  2. Open the Page Inspector. The bottom of the Inspector holds settings specific to the External Link, as shown below. The top contains general page attributes, most of which only have an effect if the External Link is set to "page within page."
