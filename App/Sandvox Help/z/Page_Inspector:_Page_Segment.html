Page Inspector: Page Segment

Publishing

Draft (do not publish) marks the page as a draft and does not publish. Not applicable to the Home Page.

Navigation

Open in new window will open that page in a new browser window (or tab).

Include page in:

Indexes & previous/next links

Uncheck the box to stop the page appearing in its parent collection's index. Not available for the Home Page.

Site maps

Choose if the page should appear in site maps. This affects both Sandvox Site Map pages and Google Sitemaps. Not available for the Home Page.

Menu As:

Check the "Menu As" box to have a link to the page appear in the Site Menu.

If you fill in the "Menu As" field, that is the title that will appear in the Site Menu. Leave it blank and the page's title will appear instead.

Meta Tags/Keywords

This is where you would put in keywords for the page, to be used by search engines.

Date/time

Sandvox should automatically set this correctly for you, but you can also set the date and time yourself.
