File Name

The file name of a page is the name of the HTML file produced when you publish the site. File names are limited to only allow the use of certain characters:

  • ASCII letters (A-Z, a-z)
  • numbers (0-9)
  • underscores and hyphens

This is because many hosts have issues with file names using characters other than these.

Setting File Names

Sandvox automatically sets a page's file name from its title. The file name is the page title with any characters not in the list above removed, with spaces replaced by underlines. If the page title consists solely of characters not in the list (for example a title in Japanese), Sandvox sets the file name to be a random-looking, unique string of letters and numbers.

If you change the title of the page, the automatically generated file name will also update itself to match the page title. Once that page has been published, however, the name no longer changes automatically. This allows you to update a title without losing any links to that published page.

You can also set a page's file name manually yourself. You might do this to make for more readable URLs, or because your host requires a certain page to have a particular file name.

To set a page's file name:

  1. Select the page in the Site Outline.
  2. Edit the file name in the Page Details area.

You cannot change the file name of a site's Home Page. If you were able to, web browsers would not be able to properly locate the Home Page when visiting the site!
