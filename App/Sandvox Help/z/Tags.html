Tags

Entering keywords/tags in the Page Details area.

Sandvox allows you to attach tags to a page, in much the same way that you can add keywords to photos in iPhoto. Keywords allow you to categorize pages based on their content.

To add keywords to a page:

  1. Select the page in the Site Outline.
  2. Click in the "Keywords" box in the Page Details area.
  3. Type a keyword such as "news" or "world news".
  4. Press the return key. The keyword becomes surrounded by a blue "lozenge."
  5. Enter more keywords if desired.

If you drag an image from iPhoto into the Site Outline to create a photo page, Sandvox will automatically take any keywords from that image and add them to the page for you.

Usage

Once you have added keywords to a page, Sandvox makes use of them in the following ways:

<meta> tags
Sandvox inserts the keywords in <meta> tags at the start of the page's published HTML. These meta tags are then used by some search engines to enhance searches of your site. While they were highly regarded a while ago, they are not particularly important. (It is not a good idea to use more than a few keywords or phrases, and the words should also appear in the page's main content.)
RSS Feeds
If the page is part of a collection that generates an RSS feed, the keywords will be included in the RSS feed. Some RSS readers and websites then make use of this information.
Spotlight
Spotlight looks at the keywords of every page of a Sandvox site when searching. For more information, please see the "Spotlight Support" article.
