Using an Embedded Image as a Link

Embedded images can be used as links just like normal text.

To turn an embedded image into a link:

  1. Click the image once to select it.
  2. Click the "Create Link" item in the toolbar, or open the inspector and go to the link segment.
  3. From the 'Link To' popup menu choose where you would like the link to go.
  4. Drag the target icon to a page in the Site Outline or enter the address of another webpage.
