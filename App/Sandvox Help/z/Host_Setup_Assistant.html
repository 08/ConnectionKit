Host Setup Assistant

The Host Setup Assistant is used to set up how a site is published. It is used both when initially publishing the site, and when making changes to the publishing process later.

Ways to open the Host Setup Assistant:

  • Click the "Setup Host" item in the toolbar.

When the Host Setup Assistant first opens, it is to the Host Overview pane.

For information on how to set up your site for publishing using the Host Setup Assistant, please see the "Setting Up Your Host" article.
