Adding music files to a site

There are a number of different ways to add music files to a site. Which method you choose depends on how you want visitors to listen to them.

Individual Pages

A music file can be placed on a site so that it occupies its own page. You link to this page and visitors can listen to the track there.

  1. Locate the music file in the Finder, Media Browser or another application.
  2. Drag and drop it into the Site Outline to create a new Video page using the file.

or:

  1. Click the "Pages" item in the toolbar and choose "Video Page." Select the new Video page in the Site Outline.
  2. Open the Page Inspector.
  3. Use the "Choose…" button to select the music file.

Then, on another page, create a link to the Video page containing the music file.

Downloads

If you wish for visitors to simply download the music file, you need to create a File Download page and link to that.

  1. Click the "Pages" item in the toolbar and choose "File Download." Select the new File Download page in the Site Outline.
  2. Open the Page Inspector.
  3. Click the "Choose…" button and select the music file.
  4. On another page, create a link to the File Download page.

This is the only way of adding an audio file to a site that doesn't require site visitors to have Quicktime installed on their computer. By adding the music as a File Download, it will either download to their computer or play using the media player of their choice.

Within a Page

A music file can also be placed within a page's main content area to allow visitors to listen to the audio while reading the rest of the page. To do so requires a Video pagelet set as a callout.

  1. Make a new Text page, Collection or Raw HTML page. You can also select an existing one of these in the Site Outline.
  2. Locate the music file in the Media Browser, Finder or another application.
  3. Drag and drop the file onto the Sidebar or main title area of the page.
  4. Select the new Video pagelet that was created by this operation and open the Selection Inspector.

  5. Set the "Location" of the pagelet to "Callout."

Alternatively, you could keep the pagelet in the Sidebar. If it appears on a Collection page, you can use this to have the music available on all pages that are part of the Collection as well.

Follow the instructions above, but set the pagelet location to "Sidebar" rather than "Callout." Check the "Show on child pages" box if desired.
