Collection Archive

A Collection Archive pagelet generates archival pages for a collection each month to keep track of changes to the collection.

If you select the collection you wish to archive before inserting a Collection Archive pagelet, the pagelet will automatically refer to, and create the archive for, the collection that was selected.

If you create a Collection Archive pagelet when you do not already have a collection selected, then you will need to specify the collection that you wish to archive in the pagelet Inspector. At the bottom of the screen, click the red target and drag to the collection you wish to archive in the Site Outline.
