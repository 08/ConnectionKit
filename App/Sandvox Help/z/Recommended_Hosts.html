Recommended Hosts

Sandvox is able to work with a large number of hosts, so any host you may have chosen will probably work just fine.
However, if you are looking for a hosting company, we can wholeheartedly recommend A2 Hosting.

Host Settings

  • Since all plans include SSH, use SFTP for the best security.
  • The document root is public_html.
