Host Settings

DreamHost

  • Please use SFTP for security
  • The Document Root is generally the same as your domain name, e.g. "foobar.com"

Live Oak Hosting

  • Please use SFTP for security
  • If using FTP, the document root is /httpdocs.
  • If using SFTP the document root is /var/www/vhosts/example.com/httpdocs. Replace "example.com" with your virtual domain. You can easily choose this directory using the "Browse" button.

Server Logistics

  • Please use SFTP for security
  • The document root is /example.com where "example.com" is your virtual domain.
  • You may be supplied with just an IP address for the server, use that as the hostname.

.Mac

  • Use the publish to .Mac option in the Host Setup Assistant
