External Link

An External Link is used to add an external webpage to your site. The External Link appears in the Site Outline like any other page, but acts only as a "placeholder" for the external webpage. You can then link to the External Link in the Site Outline.

By default, if you have it set in your Preferences, new External Links point to the current web page open in Safari.

Settings

To edit an External Link:

  1. Select the External Link in the Site Outline.
  2. Open the Page Inspector. The bottom of the Inspector holds settings specific to the External Link, as listed below. The top contains general page attributes, most of which only have an effect if the External Link is set to "page within page."

Full URL of external page

The address of the external page. e.g.

http://www.apple.com/

Type of Link

Simple link to page
The External Link behaves as a normal link.
Open page in new window
The external page opens in a new web browser window.
Page within page
The external page is shown inside of your site. An example of this is shown below. You can specify the height of the area the page is drawn in. When using this option, all general page attributes are available.
