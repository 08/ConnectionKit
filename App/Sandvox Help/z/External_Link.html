External Link

An External Link appears in the Site Navigator much like a regular page, but acts only as a "placeholder" for an external webpage. This can be useful for adding it to the Site Menu, a Site Map, or index.

Ways to add an External Link to a site:

  • Use the "New" item in the toolbar.
  • Choose "External Link" from the Insert menu.
  • Drag a website's URL into the Site Navigator.

Settings

To edit an External Link:

  1. Select the External Link in the Site Navigator.
  2. Go to the page details area at the bottom of Sandvox, there you can edit the URL.

The window title, and the meta descriptions are automatically pulled from the website you link to.

Full URL of external page

The address of the external page. e.g.

http://www.apple.com/
