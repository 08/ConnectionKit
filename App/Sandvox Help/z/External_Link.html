External Link

This page describes the use of the External Link page, not the External Page Object. If you simply want to add a link within another type of page (such as a text page or photo page), please see the Link article.

You can add an External Link (a link to a web site other than your own) on an External Link page. An External Link page appears in the Site Outline like any other page, but acts only as a "placeholder" for the external webpage. You can then link to the External Link in the Site Outline.

By default, if you have it set in your Preferences to "Read URL's from web broswer", when you create a new External Page, Sandvox will read and insert the URL from the current window of your browser (for Safari, Opera, Camino, Firefox, and OmniWeb).

Ways to add an External Link to a site:

  • Use the "New" item in the toolbar.
  • Choose "External Link" from the Insert menu.
  • Highlight and drag the URL of the site you wish to incorporate from your web browser to the Site Outline.

Settings

To edit an External Link:

  1. Select the External Link in the Site Outline.
  2. Go to the page details area at the bottom of Sandvox, there you can edit the URL.

The window title, and the meta descriptions are automatically pulled from the website you link to.
