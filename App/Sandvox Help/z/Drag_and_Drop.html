Drag and Drop

Sandvox has extensive drag and drop support. For example, using drag and drop, you can easily create a new page or object.

The table below lists the various items you can drop in the Site Navigator and the page that will result.

Dragged Item Result
HTML file from the Finder Raw HTML page
Video, audio or Flash file Video page
Image from the Media Browser, Finder or another application Photo page
Text selection or text file from the Finder Raw HTML/Text page
Any other type of file from the Finder File Download
URL or Safari bookmark External Link Page
Item from Delicious Library External Link to the item's Amazon page


Or, if you drag an item into the Web View:

Dragged Item Result
HTML file from the Finder Raw HTML object
Video, audio or Flash file Video object
Image from the Media Browser, Finder or another application Photo object
URL or Safari bookmark Link to the URL
RSS feed URL RSS Feed page
Item from Delicious Library Photo object of the item's thumbnail. The object then links to the item's Amazon page
