Drag and Drop

Sandvox has extensive drag and drop support. For example, using drag and drop, you can easily create a new page or pagelet.

The table below lists the various items you can drop in the Site Outline and the page that will result.

Dragged Item Result
HTML file from the Finder Raw HTML page (requires Sandvox Pro)
Movie, audio or Flash file Movie page
Image from the Media Browser, Finder or another application Photo page
Text selection or text file from the Finder Text page
Any other type of file from the Finder File Download
URL or Safari bookmark External Link
Item from Delicious Library External Link to the item's Amazon page


Or, if you drag an item into the Web View:

Dragged Item Result
HTML file from the Finder Raw HTML pagelet (requires Sandvox Pro).
Movie, audio or Flash file Movie pagelet.
Image from the Media Browser, Finder or another application Photo pagelet.
Text selection or text file from the Finder Text pagelet or inserts the text into an existing piece of text.
URL or Safari bookmark Link to the URL.
RSS feed URL RSS Feed Pagelet.
Item from Delicious Library Photo pagelet of the item's thumbnail. The pagelet then links to the item's Amazon page.
