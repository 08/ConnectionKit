Wrap Inspector

This sets where an object is placed in the text.
You set the position of an inline object (how text flows around it) by checking the 'object causes wrap' box and then clicking the box that illustrates how you would like the text to wrap.
Dragging an object near the left, center and right sides of the main body will automatically set the alignment (from the first 3 choices); turning off object causes wrap allows an image to located directly within the text, which is great for small graphics...
You also move objects to a callout or the sidebar by selecting the the option in the inspector.

- "introduction" and "title" can only be turned on when placed as a callout or in the sidebar
- "callout" is the only to make text flow around a non-image object

ONLY images and Raw HTML Objects can have "object causes wrap turned off" -- other objects have this on always.
