Thumbnail

The thumbnail is used in a number of contexts, such as the Site Outline, for other pages that index the contents of a page, such as a Photo Grid, and for an image representing a page if it is to be shared with Facebook.

For any page with a thumbnail Sandvox displays the page's thumbnail in the Site Outline. This will help you recognize many pages with just a glance. The home page does not get a thumbnail, so it is disabled.

To set a page's thumbnail:

  1. Select the page in the Site Outline.
  2. Open the Page Inspector to the Appearance tab.
  3. From the "Thumbnail" popup menu, choose image or custom image.
  4. "Image" will use an image that is on the currently selected page in the site outline. "Custom" will open a Finder window so you can navigate to your chosen image. If you have chose "Custom" you can also drag in image into the "thumbnail" preview box.

Sandvox automatically sets the thumbnail from the page's content, looking for the most dominant image on a page. (This only applies to local media, not media loaded up over the Internet; you will need to create a thumbnail yourself.)

To reset or remove a page's thumbnail:

  1. Select the page in the Site Outline.
  2. Open the Page Inspector to the Appearance tab.
  3. From the "Thumbnail" popup menu, select "None"
