Embedding YouTube video in Sandvox

Note: This article requires the Pro edition of Sandvox.

With Sandvox Pro, it is fairly easy to add a YouTube video to your site. First you need to get hold of the code that YouTube provides for embedding video.

  1. Use the YouTube website to locate the video you require.

  2. Next to the video you should see this box:

  3. Copy all of the code in the box to the clipboard.

Then, you add the code to your desired location in Sandvox.

To add the video to part of a Text page:

  1. Select the page in the Site Outline.
  2. Click some of the existing text to begin editing it.
  3. Choose "Edit Raw HTMLâ€¦" from the Edit menu.
  4. Paste the YouTube code you copied into a suitable place. Be sure not to paste it in the middle of a tag!
  5. Close the HTML editing window.

You can use this basic process to insert the YouTube code almost anywhere in a site; just make sure the video will fit! For example, if you intend to create a pagelet with the video in, you will want to change the dimensions to be roughly 200 x 160 pixels.
