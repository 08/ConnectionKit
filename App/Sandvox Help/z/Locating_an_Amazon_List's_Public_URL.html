Locating an Amazon List's Public URL

An Amazon List can take its content from an existing list at Amazon's website. The Amazon List pagelet supports:

  • Wish lists
  • Listmania! lists
  • Wedding Registries (Only for the US and UK stores)

Every list has a unique ID number and a URL from which other people can view the contents of the list. You can enter either of these into the "List" field of the pagelet inspector for an Amazon List.

Making the List Searchable

To use a wish list from Amazon's website, it must be marked as "public" or "searchable" otherwise Sandvox cannot load it. This does not apply to Listmania! or wedding registries; they are inherently public. Slightly different instructions apply for each store.

Instructions for making your wish list searchable:

  • US
  • UK
  • Canada
  • France
  • Germany
  • Japan

Finding the URL

You now need to find the actual public URL of the list.

For a Wish List:

  1. Go to the Amazon store's website.
  2. From the top of the page click the "Wish List" link.
  3. There will be a search box for locating somebody's Wish List. Enter your name or e-mail address into it to search for your list. This step is essential!
  4. If Amazon does not jump straight to your Wish List, select it from any search results that appear.
  5. Copy the URL of the list's webpage from the address bar and paste it into the Sandvox Inspector.


For Listmania! and Wedding Registries:

  1. View the list in your web browser.
  2. Copy the URL of the webpage from the address bar and paste it into the Sandvox Inspector.
