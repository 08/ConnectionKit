General Page Attributes

The Page Inspector is split into three tabs. The first tab is the 'page' tab and contains settings that are applicable to publishing, navigation, meta tags / keywords, and the time and date. The second tab is the 'appearance' tab, that is where you can choose the page layout and sidebar items. The third tab is collection specific.

Page Tab

Publishing

Draft (do not publish) marks the page as a draft and does not publish. Not applicable to the Home Page.

Navigation

Open in new window will open that page in a new browser window (or tab).

Include page in:

Indexes & previous/next links

Uncheck the box to stop the page appearing in its parent collection's index. Not available for the Home Page.

Site maps

Choose if the page should appear in site maps. This affects both Sandvox Site Map pages and Google Sitemaps. Not available for the Home Page.

Menu As:

Check the "Menu As" box to have a link to the page appear in the Site Menu.

If you fill in the "Menu As" field, that is the title that will appear in the Site Menu. Leave it blank and the page's title will appear instead.

Meta Tags/Keywords

This is where you would put in keywords for the page, to be used by search engines.

Date/time

Sandvox should automatically set this correctly for you, but you can also set the date and time yourself.

Appearance Tab

Layout

Page Title

Uncheck this box if you do not want the pages title to show on the page.

Comments

Check this box if you want to allow user comments on the page. You also need to configure the comments.

Date Created, Modified

Check the box to display the page's date and time of creation, or modification.

Show sidebar

Hide or show the sidebar on this page by checking or unchecking the sidebar box. You can also choose to include or remove specific objects by checking or unchecking them in the sidebar box.

Thumbnail

To select a custom image, select 'custom image' from the pop up menu and navigate to the image in the finder window.
