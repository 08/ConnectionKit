System Requirements

To run Sandvox on your computer requires:

  • A PowerPC or Intel Macintosh computer. (Sandvox is a Universal Application).
  • Mac OS X "Tiger" version 10.4.4 or later. Sandvox does not run on OS X 10.3 (Panther) or 10.2 (Jaguar), nor does Karelia have plans to support these earlier versions of OS X.
  • To use graphical text requires a Quartz Extreme-capable Mac. Please see this Apple Support article for details.


Sites published by Sandvox are viewable with almost any web browser, although there are two small caveats:

  • Graphical text and PNG images are not properly viewable in Internet Explorer 6 and earlier due to an Internet Explorer bug. To compensate, graphical text is replaced with plain text for visitors using Internet Explorer 6 and earlier.

    Please see "My site looks different when viewed with some web browsers" for further details. Both Graphical text and PNG images will appear correctly in Internet Explorer 7 and up.

  • If a site visitor's web browser does not support CSS (all modern web browsers support CSS), only the textual content and basic images from the site will appear.
