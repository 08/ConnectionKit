System Requirements

To run Sandvox on your computer requires:

Overview

  • An Intel or PowerPC Macintosh computer. (Sandvox is a Universal Application).
  • Mac OS X 10.5 "Leopard," or Mac OS X 10.6 "Snow Leopard." Sandvox is not compatible with earlier versions, including 10.4 (Tiger), 10.3 (Panther), or 10.2 (Jaguar).
  • To use graphical titles Sandvox requires a Quartz Extreme-capable Mac. Please see this Apple Support article for details.
  • An internet connection fast enough to handle the amount of data you will be uploading. (In other words, we recommend that you have a good, fast Internet connection if you are going to be publishing a large website.)

Operating System Details

Mac OS X 10.5 "Leopard"
All features of Sandvox are available.
You may experience a crash when using the full-screen previews from the "Discover Plugins" window. Apple fixed this in Mac OS X 10.6.
Mac OS X 10.6 "Snow Leopard"
All features of Sandvox are available.
Mac OS X 10.7 "Lion"
Sandvox has not yet been certified to work with pre-release versions.

Browser Compatibility

Sites published by Sandvox are viewable with almost any web browser, although there are two small caveats:

  • Graphical Titles and PNG images are not properly viewable in Internet Explorer 6 and earlier due to an Internet Explorer bug. To compensate, graphical titles are replaced with plain text for visitors using Internet Explorer 6 and earlier.

    Please see "My site looks different when viewed with some web browsers" for further details. Both graphical titles and PNG images will appear correctly in Internet Explorer 7 and up.

  • If a site visitor's web browser does not support CSS (all modern web browsers support CSS), only the textual content and basic images from the site will appear. It may be helpful to include a link on your site to the latest version of Internet Explorer to allow visitors to download it so that your page may be viewed as intended.
