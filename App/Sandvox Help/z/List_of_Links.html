List of Links

A List of Links is a pagelet that holds a list of hyperlinks to external webpages. A typical List of Links looks something like this:

You might use a List of Links to show visitors other sites of yours, or sites that you are interested in.

Editing

To edit a List of Links, you use the Selection Inspector. In addition to the parameters listed listed below, there are some general pagelet attributes that can be set.

Don't forget that you can also click the Editing Marker () to insert additional text into the pagelet.

To add a link, click the plus button in the Inspector. If you have it set in your Preferences, the new link will be to the frontmost site in Safari.

To remove a link, select it in the table and click the minus button.

To edit a link, select it in the table and use the boxes below the table to enter the title and URL of the link.
