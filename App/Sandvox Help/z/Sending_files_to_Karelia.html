Sending files to Karelia

In some cases, the support team at Karelia software may ask you to send a copy of your Sandvox document (or some other supplemental files) to us.

Zip and rename the zipped file

  1. Compress your Sandvox document into a "zip" file by control-clicking (right-clicking) on the document and choosing "Compress <filename>".
  2. Rename this zipped-up file to the case # assigned to your case by Karelia support so we can keep track of it.
  3. Send the file to us. Here are some ways to do that:

If it is a small file

Less than 10 MB in size, reply to the email from Karelia support and attach the zip file to it.

If you use DropBox

Drag the zip file into your Public Folder, as described here: <http://www.dropbox.com/help/16>.
Then reply to the email from Karelia support with the public link for your file so we can grab it.

Using our free 'Upload to Karelia' utility

  1. http://www.karelia.com/files/Upload_to_Karelia.zip
  2. Unzip this, if needed, and then launch it.
  3. Drag your zipped file onto its window.
  4. Wait for the uploader to finish; it may take a while.
  5. Then reply to the email from Karelia support so we know the file has been uploaded.

Try an online file transfer service

  • https://www.transferbigfiles.com/
  • https://www.yousendit.com/
  • Try a Google search for more
