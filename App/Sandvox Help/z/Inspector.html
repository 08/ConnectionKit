Inspector

The Inspector is used to edit a site in Sandvox. It allows you to set various properties for the current site, page and selection. The Inspector uses 6 segments they are: Document, Page, Wrap, Metrics, Link, and Object.

Ways to show or hide the Inspector:

  • Click the "Inspector" item in the toolbar.
  • Choose "Show Inspector" or "Hide Inspector" from the View menu.
Document segment (Shortcut: Command-1)
Contains settings that affect the entire site
Page segment (Shortcut: Command-2)
Contains setting that affect the page you have selected in the site outline
Wrap segment (Shortcut: Command-3)
Has Object Placement for placing objects as Inline, Callout, or in the Sidebar. There are options for the Layout, you can also specify a Border for your objects (if allowed by the design)
Metrics segment (Shortcut: Command-4)
Used to adjust Width and Height of an image to the pixel
Link segment (Shortcut: Command-5)
Use the Link To to create links to internal pages, full size images, or external URL's
Object segment (Shortcut: Command-6)
Use this segments to setup the setting specific to the object you have selected
