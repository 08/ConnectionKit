Licensing

Sandvox is available in both Normal and Pro forms, where Sandvox Pro costs a little more. For more information, please see "Sandvox Pro."

Additionally, you can choose the number of users to license Sandvox for:

Single User
Allows you to install Sandvox either:
  • On a single computer for use by any number of people.
or
  • On multiple computers for non-concurrent use by yourself only.
Household
Can be installed on up to four computers whose primary location is in a single household. This does not extend to commercial users of Sandvox.
Site License
A discounted license that allows you to install Sandvox on as many computers as you purchase a license for. You must purchase a minimum of five seats.
