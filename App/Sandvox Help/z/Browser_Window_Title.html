Browser Window Title

The Browser Window Title is the text that appears at the top of a visitors web browser when viewing your site:

The same text also appears in the titlebar of Sandvox when working on a site.

The Browser Window Title is composed of the Page Title, Site Title and Site Author. If the "Author" field in the Site Properties is blank (i.e. the site has no author), then the Site Author does not appear in the Browser Window Title.

Advanced Tip

You can use Sandvox's Hidden Preferences to set the separator used between the components of the Browser Window Title.
