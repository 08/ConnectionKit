Tweet Button

The Tweet Button is a small object which allows users to easily share one of your web pages with their followers on Twitter
If you are looking for a way to share your Twitter feed on your website, please see the Twitter article.


You can customize the look of your Tweet button in the Object Inspector

Appearance

Vertical Count
Shows the count above the tweet button
Horizontal Count
Shows the count next to the tweet button
No Count
Do not show a count, only the tweet button

Tweet

Text
Is the text of your tweet.
URL
Is the URL of your web page.
Twitter Account
This is you Twitter account name.

The steps a user goes through when using the Tweet Button are:

  1. The user clicks the Tweet Button
  2. The user is asked to login to Twitter if they arenâ€™t already. If the user is new to Twitter they can also create an account.
  3. The Share Box appears already completed with the information provided in the properties of the Tweet Button. The user can change the content if they wish.
  4. Posting of the Tweet is confirmed and the user is suggested a maximum of two accounts they may wish to follow as provided in the properties of the Tweet Button.
  5. The Share Box remains open until the user presses close.
