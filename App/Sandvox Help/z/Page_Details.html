Page Details

The Page Details area is located at the bottom of the page. From it, you can edit the selected page's:

  • File Name and Extension
  • Window Title
  • Meta Description


What these fields all have in common is that they are all essential content on a page that is not directly visible as part of the page content in the web view. When used properly and in conjunction with appropriate content, these fields can impact each page's rankings and attractiveness to search engines indexing your website's content and to people finding your pages in search engine listings.
