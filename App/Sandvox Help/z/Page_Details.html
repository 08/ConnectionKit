Page Details

The Page Details area is located at the bottom of the page. From it, you can edit the selected page's:

  • File Name and Extension
  • Window Title
  • Meta Description

When used properly, and in conjunction with appropriate content, these fields can improve a page's rankings and attractiveness to search engines, helping people find you.
