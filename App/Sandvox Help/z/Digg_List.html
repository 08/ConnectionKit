Digg List

A Digg list is a pagelet that displays a list of stories from Digg. A typical Digg list looks something like this:

Editing

To edit a Digg list, you use the Selection Inspector. In addition to the attributes listed here, there are some general pagelet attributes that can be set.

Also, don't forget you can click an Editing Marker to insert some additional text into the pagelet.

Stories by user

The first radio button marked "Stories" allows you to select stories based on the behavior of specific Digg users (including yourself). You can choose to list topics that are popular with, submitted by, dugg by, or commented on by the user whose username you enter.

If you are a frequent Digg user, one good use of this feature would be to set the Digg list to show only stories dugg by yourself, and visitors to your site will be able to see what news you are interested in.

Stories by topic

The second radio button marked "Stories" allows you to select stories based on topic. You can choose to display all stories in a particular topic, or only the most popular stories, and you can choose from many different categories and specific topics.

Maximum

Determines how many Digg items will be displayed.

Show Digg count

The number of diggs each story has received will be displayed after the link.

Show descriptions

If checked, the descriptions attached to each story are also displayed. This will make the pagelet take up considerably more room.

Open links in new window

If checked, this option will cause a new browser window to open when a link to a Digg story is clicked on.
