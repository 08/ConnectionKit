Text Color

With Sandvox you can set the color of almost any piece of text in a site. The main exceptions are links and certain titles. These colors are defined by a site's design and so cannot be easily modified.

To set the foreground color of some text:

  1. Select the text.
  2. Open the colors palette by choosing "Colors" from the Format menu.
  3. Select a color from the palette. As you adjust the palette, the text color changes in real-time.

or:

  1. Select the text.
  2. Open the Font panel by choosing "Show Fonts" from the Format menu.
  3. Click the "Text Color" button: The colors palette opens.
  4. Select a color.

You can also set the background or highlight color of text.

To set the background color of some text:

  1. Select the text.
  2. Open the Font panel by choosing "Show Fonts" from the Format menu.
  3. Click the "Document Color" button: The colors palette opens.
  4. Select a color.
