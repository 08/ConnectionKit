Block Quote

Text Boxes have an option to display as a block quote. This offers the benefits of:

  • Designs can supply custom styling/formatting to make it clear that the text is a quote
  • Sandvox generates the correct <blockquote> HTML, so, for example, screen readers can understand the nature of the text

To create a new Block Quote:

  1. From the "Objects" toolbar item, select "Block Quote"
  2. Double-click the quote to begin editing it

To turn an existing Text Box into a Block Quote:

  1. Select the Text Box
  2. Open the Object Inspector
  3. Check the "Block Quote" box
