Host Overview

Introduction

This article is part of the "Setting Up Your Host" procedure. The Host Overview is the first step of this process.

The Host Overview is designed to give you a brief summary of how Sandvox is currently set up. After you have set up the host for your site, Sandvox shows you the Host Summary. This contains the same information as the Host Overview.

Using the Host Overview

To see your host's overview, you open the Host Setup Assistant, the Host Overview is the first thing that appears.

If you have not yet set up your host, you are presented with this:


If you have previously set up the site's Host, you are instead presented with this:

In the case above, the Overview provides a quick summary of how your site will be published.

Next Step

If you have not yet set up your host, click the "Edit" button to continue to "Choosing a Publishing Location."

If you have set up your host, then you can click the "New Configuration" to enter your host settings again to continue on to "Choosing a Publishing Location."

To simply edit your existing settings instead, click the "Edit" button. Depending on how your host is set up, you will be taken to one of:

  • Publishing to your Computer
  • Publishing to .Mac
  • Publishing to a Remote Host
