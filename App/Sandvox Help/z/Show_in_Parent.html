Show in Parent

Every collection has a "show in parent" attribute. This controls how the collection will appear in its parent collection's index.

But what about the collection that contains the two photo albums? What should that page look like? That is what the "show in parent" setting is for. You select how each of the photo albums is displayed in its parent collection.

To set what appears in a collection's parent:

  1. Select the collection in the Site Outline.
  2. Open the Page Inspector.

  3. Make your choice from the "Show in parent" popup button.

The choices are:

Automatic
The collection's thumbnail and text
First item
The first page in the collection
Most recent item
The page in the collection with the most recent date
Item list: most recent first
Lists all pages in the collection, the most recent first
Item list: alphabetical
Lists all pages in the collection in ascending alphabetical order.


When using either of the two "list" options, Sandvox limits the number of titles that can be shown to 5. You can change this in the hidden preferences if you wish.
