Text

Note: This article is on Text pages and pagelets, not editing of text in general. For information on that subject, please see the "Text Editing" article.

Text pages and pagelets are very important in Sandvox. They consist of a block of text which you can edit, format and insert images into. If you are working with a text page, you can also place pagelets in the text as callouts.

A simple Text page with a callout.
A Text pagelet.

Editing

There are no specific attributes for a Text page or pagelet, but there are some general page attributes or general pagelet attributes that can be set.

For more information on editing the text of a Text page or pagelet, please see the "Text Editing" article.

Importing

Sandvox is able to create Text pages by importing some file types. At present the supported formats are:

  • Plain Text (.txt)
  • Rich Text (.rtf)
  • Microsoft Word (.doc)

To import a file, drag it into the Site Outline and Sandvox will create a new Text page from the file's contents.

Note that not all Microsoft Word format versions are supported (Microsoft tends to change the format over time). We recommend that you first open Word files with Apple's TextEdit application and save them in Rich Text format (.rtf) and then import that into Sandvox.
