Text

Note: This article is on Text objects, not editing of text in general. For information on that subject, please see the "Text Editing" article. You can type text on every page, but you can also insert a separate "Text Box" object.

Text objects are central to most web sites created using Sandvox. They consist of a block of text which you can edit, format, and insert images into. If you are working with a text page, you can also place objects in the text as callouts.

Ways to add a Text Object to a site:

  • Use the "Text Box" item from the Insert menu or the Objects toolbor.
