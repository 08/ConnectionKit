Sidebar

The sidebar is a column to the side of a page which acts as a "container" for a number of objects. It is 200 (or more) pixels wide. Depending upon the design selected for a site, the sidebar is either on the left or right:

The "Aqua" design has the sidebar to the left.
The "Rounded Blue" design has the sidebar to the right.


The sidebar is typically used to display information supplemental to the main content of the page. Often, the same objects appear in all pages of your site, giving it continuity.

For example, you might use the sidebar to provide a simple Contact Form on every page, or a List of Links to your favorite sites. For information on adding and removing objects from the sidebar, please see the "object" article.

To set whether or not the sidebar appears on a page:

  1. Select the page in the Site Outline.
  2. Open the Page Inspector, to the Appearance tab.
  3. Check or uncheck the "Sidebar" box.

The pages of a site are arranged in a hierarchy, with collection pages "containing" other pages. You can set whether the sidebar of a page should include those objects that appear in its parent collection.

To set whether or not the sidebar should include objects from its parent page:

  1. Select the page in Site Outline.
  2. Open the Page Inspector.
  3. Check or uncheck the box that contains the specific object.
