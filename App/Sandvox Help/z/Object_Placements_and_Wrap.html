Object Placements and Wrap

To move a media placeholder object:

  1. Click the image to select it. It will have a box around it .
  2. Drag the image to a new location within the text.

This sets where the image is placed in the text. You can then also set the position of the image (how text flows around it) in the wrap inspector by checking the 'object causes wrap' box and then clicking the box that illustrates how you would like the text to wrap.
