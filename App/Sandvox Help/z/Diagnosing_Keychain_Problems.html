Diagnosing Keychain Problems

When publishing, some users have reported issues with publishing where the password required for publishing cannot be retrieved from the keychain.

Here are some techniques which might help.

  • Launch your "Keychain Access" utility. Then lock your keychain by clicking the lock icon in the upper left corner of the Keychain Access window. This forces the system to request authentication, which can often unstick the problem and allow Sandvox to find the requested password for publishing.
  • Delete or move your preferences file. The file is found at home folder → Library → Preferences → com.karelia.Sandvox.plist. Drag this file to your Desktop (or the Trash, if you don't mind permanently resetting your Sandvox Preferences). Starting with empty Sandvox Preferences, rather than reinstalling, has also been reported to help unstick the problem and allow Sandvox to find the proper keychain entry.
  • As a last resort, it may also be worth using Keychain Access's "Keychain First Aid" command to verify and repair the integrity of the default keychain.
