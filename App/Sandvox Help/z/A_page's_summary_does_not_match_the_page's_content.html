A page's summary does not match the page's content

Sandvox has a feature which allows you to specify a custom summary for a page. WIth this enabled the summary of the page is "disconnected" from the actual page content. The summary then does not update itself to reflect the page's content.

To make a custom summary automatic again:

  1. Control-click a part of the summary text in a Collection's index.
  2. Choose "Remove Custom Summary of Pageâ€¦" from the contextual menu.
