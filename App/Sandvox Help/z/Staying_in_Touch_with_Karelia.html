Staying in Touch with Karelia

In order to help keep you up to date with the latest versions and news about Sandvox, the application offers you two ways to stay in touch. When you first launch, you are asked how you would like to be kept updated.

Application Update Notifications

Sandvox can periodically check to see if there is a new version available. If you check this option when the application is first launched, Sandvox will automatically check for updates, up to once per day. If a new version is available, you can download and install it right away.

You can change the option to check for updates from the Preferences window, either disabling or even choosing a feed that will notify you of any cutting-edge "beta" releases.

Regardless of your preferences, you can manually check for updates from the application menu.

Note: If an Internet connection is not available when you sign up, your information will be saved and sent to Karelia the next time Sandvox is launched with an Internet connection available.
