Replacing the Home Page with an alternative page type

This article is aimed at advanced users of Sandvox only. As such, any modifications to Sandvox using the techniques described below cannot be supported by Karelia.

Usually, Sandvox uses a plain Text page for the Home Page. However, in some circumstances, you may wish to change this. This can be achieved using Sandvox's Hidden Preferences, but only for a new site, not an existing one.

To create a site with a different Home Page type:

  1. Quit Sandvox and open the Terminal.
  2. Type the following command: 
    defaults write com.karelia.Sandvox DefaultRootPageBundleIdentifier "sandvox.PhotoPage"
    Replace "sandvox.PhotoPage" with the bundle identifier of the page type you wish to use.
  3. Hit Enter and relaunch Sandvox.
  4. Create a new site. It should have your new page type choice as the Home Page.

This setting will now affect all new sites you create. If you only want to make the change for just this one site, you need to reset Sandvox back to its default.

To reset the Home Page type:

  1. Quit Sandvox and open the Terminal.
  2. Enter the following command: 
    defaults delete com.karelia.Sandvox DefaultRootPageBundleIdentifier
