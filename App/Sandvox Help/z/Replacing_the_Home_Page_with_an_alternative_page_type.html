Replacing the Home Page with an alternative page type

Usually, Sandvox uses a plain Text page for the Home Page. If you want the home page of your site to be a different type, you can make that choice when you first create the site. Once a site is created, you can no longer change the page type of the home page.

Setting the page type of the home page when creating a new site is simple.

To set the page type of the home page for a new site:

  1. From the File menu, select "New Site..."
  2. Under "Home page type," select the page type you prefer for the home page.
