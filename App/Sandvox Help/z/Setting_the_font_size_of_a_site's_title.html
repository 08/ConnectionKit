Setting the font size of a site's title

Sometimes a site you are working on has a long or short title that doesn't quite fit into the available space as well as you'd like. How you set the size of the font used for the title depends on if the site's design uses graphical text.

To set the font size of a non-graphical site title:

  1. Select the Home Page in the Site Outline.
  2. Click the site title to begin editing it.
  3. Select all of the title text.
  4. Choose "Show Fonts" from the Format menu to open the font panel.
  5. Set the font size and close the panel.
