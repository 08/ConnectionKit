Favicon

Every site has a favicon. This is a small image (16x16 pixels) that is used to represent your site. The icon appears in your web browser when viewing the site. For example, in Safari, the favicon for a typical site is found here:

Also, while working on a site in Sandvox, the favicon is displayed next to the Home Page in the Site Outline:

To set a site's favicon:

  1. Open the Site Inspector.
  2. Go to the Media tab.

  3. Drag an image from the Media Browser, the Finder or another application to the "Favicon" box.

To remove the icon again, click the cross icon next to the box.

Once a site is published, the favicon is 16x16 pixels. If you try to set a larger image for the favicon, Sandvox will automatically resize it for you. However due to the tiny sizing, this may result in a severe loss of quality, so using a 16x16 pixel image to start with is advised.
