Photo Grid

A photo grid index is a great way to show off the most recent additions to your Photo Album.

To add a photo grid index

  1. Choose the page that you would like to add the photo grid to in the site outline.
  2. Add the Object by choosing "Objects" from the toolbar, then "Indexes", the "Photo Grid".
  3. Open the Object Inspector, choose the newly added photo grid if it is not already selected.
  4. Drag the little target to the photo album that you would like to use in the site outline.

You can limit the number of images show by checking the "show at most" box and choosing the number that you want shown.

Gallery

The gallery option will create a slideshow of your images.
When a gallery is activated, that visitors will not be taken to the individual page containing the photo, but will just see the image from that page, no other content that might be on that page. The gallery is not for viewing videos or other content, just images.
You can choose the background colors and the transition.
Check the 'Wrap around' box to have the images continually play.
You can choose to have the slideshow start automatically or start manually.
And with the speed slider you can choose the speed that the images are shown.
