Image Sizing

Introduction

To set an image size with the Inspector open the Inspector to the Metrics tab. Use the size height and width boxes to set the number pixels high or wide you want your image.
Checking the "constrain proportions" box will keep the image height/width proportionate.

To set an image size in the page, click the image to select it and drag the small boxes in the corners to make the image larger or smaller.

Text Wrap

You can use the Wrap tab in Inspector to select the Object Placement or check the box for Object causes wrap.
To move the image up or down on the page simply click and drag it.

Linking to Larger Images

Independent of which photo image size you set, you can still link to a larger, full-size version of your image.

To link to the full-size version of a Photo image:

  1. Select the Photo image.
  2. Open the Inspector to the Link tab.

  3. In the "Link To" pop up and select "Full-size image."
