Linking To External Pages

There are a number of ways of linking to an external page, depending upon what you want to achieve.

Link
The easiest and most common method. A simple "clickable" link within a text area.
External Link Page
A page in the Site Outline that acts as a "placeholder" for the external page. You then link to this from other pages.
List of Links
A pagelet containing a list of links to external pages that you specify.
Embedded Page
Displays the content of the external page inside of a pagelet. Generally only suitable for narrow pages.
