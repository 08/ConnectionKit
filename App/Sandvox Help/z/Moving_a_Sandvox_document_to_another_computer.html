Moving a Sandvox document to another computer

To move your document to another computer, you can generally treat it like any other file, copying the .sandvox file over.

Normally, Sandvox documents contain all the media your site relies on, so they'll transition across to the new computer just fine. However, individual videos, audio and Flash files can be configured in the Object Inspector not to "Copy file into document". If you've done this at all, it's probably wise to reverse that.

For each object:

  1. Select it
  2. Open the Object Inspector
  3. Check the "Copy file into document" box

Then:

  1. Save the document and close it
  2. Proceed to move/copy the file to the new computer
