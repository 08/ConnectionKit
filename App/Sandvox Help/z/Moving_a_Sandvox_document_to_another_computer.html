Moving a Sandvox document to another computer

Moving a site for editing to another computer requires careful attention. Because a Sandvox document is actually a package of files, how you copy it depends on whether that package includes copies of media files you have inserted or not.

What you need to do depends on how the "keep media originals with document" setting (see the "Media" article for details) was set for the site when you first began working on it. (Please note: Changing this setting after the site has been created will not effectively change the way the site needs to be copied in order to move it. Use the method below that suits the way the setting was originally selected.)

Copy all files
If you've been using "Copy all files," moving your site is easy. Simply copy the document to the new computer; everything should remain intact.
Automatic
The name of your Home Folder must be the same on both machines. Copy the document and any media files used from the Pictures, Movies and Music folders to the new computer.
Reference all files
The name of your Home Folder must be the same on both machines. Copy the document and all media files used to the new computer.

Of course once you have finished the move, it is a good idea to check through the site on the new computer to make sure no media files are missing.
