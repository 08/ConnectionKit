Moving a Sandvox document to another computer

Moving a site for editing to another computer can be tricky. It all depends on how the "keep media originals with document" setting (see the "Media" article for details) was set for the site when you first began working on it:

Copy all files
Simply copy the document to the new computer, everything should remain intact.
Automatic
The name of your Home Folder must be the same on both machines. Copy the document and any media files used from the Pictures, Movies and Music folders to the new computer.
Reference all files
The name of your Home Folder must be the same on both machines. Copy the document and all media files used to the new computer.

Of course once you have finished the move, it is a good idea to check through the site on the new computer to make sure no media is missing.
