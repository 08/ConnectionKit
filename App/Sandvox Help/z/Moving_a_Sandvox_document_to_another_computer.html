Moving a Sandvox document to another computer

To move your Sandvox site for editing to another computer, you would copy the .sandvox file and move it to the other computer.


If while building your Sandvox document you have changed any of your media (movies / music files) settings so that they are no longer saved in the document you will need to go and copy those files into the Sandvox document. To do that:

  1. Check the "copy file into document" check box for any media where you'd previously unchecked it.
  2. Save the doc
  3. Close the doc and move the file
