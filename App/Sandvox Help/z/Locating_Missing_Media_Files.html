Locating Missing Media Files

In certain circumstances, it is possible for a media file you have placed in your site to turn up missing.

This is most likely to happen when you have told Sandvox to reference media files rather than copying them and then you move the file. Sandvox will continue to look for the file in its original location.

If this happens, the next time you open your site in Sandvox, you will be prompted to search for the missing media file(s). A screen like the one below will appear, telling you the name of the image and where it was when you inserted it into Sandvox. If Sandvox is still able to locate a thumbnail image of the missing media, that will also be displayed in the second column to help remind you what image is missing.

If you know that you have moved the file, you can click the "Find" button to navigate to its new location.

If on the other hand, you have deleted the file because you no longer need it, you can click "Continue" in order to continue opening Sandvox--but remember to delete the reference to the media file in your site, so that visitors won't find items missing.

Note that you can usually avoid missing media files by directing Sandvox to copy media files. For more information, please see the Media article.
