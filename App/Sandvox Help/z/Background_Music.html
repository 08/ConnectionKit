Background Music

You can easily add background music to a single page or a whole collection of pages in your site.

To add background music:

  1. Add a new Media Placeholder object to the pages you wish to have the background music. You can add the Media Placeholder to a Collection page (such as the Home Page) in order for the music to play on every page in the collection.
  2. Select the new media object.
  3. Open the Object Inspector and set the file info to use the appropriate file for background music.

If you have added the Media object to a Collection page in the sidebar, it will show on all of the pages of that collection that are also showing the sidebar.

To have the background music automatically play when the page is opened, check the "Automatically Play" box in the Object Inspector.

You may wish to have the background music play continuously. To do so, check the "Loop" box in the Object Inspector.
