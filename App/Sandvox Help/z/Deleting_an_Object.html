Deleting an Object

When deleting an object you remove it from a page.

To delete an object:

  1. Click the object to select it.
  2. Choose "Delete" from the Edit menu.

You can also click the object and hit the delete key on your keyboard.

You can undo this by choosing "Undo" from the Edit menu.

If removing an object from the main page of a collection it will be removed from the collection. You can remove it from pages within the collection on the individual pages.
