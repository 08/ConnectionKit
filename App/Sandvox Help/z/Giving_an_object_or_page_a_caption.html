Giving an object or page a caption

You can give almost any kind of object a caption. This is a short piece of text below the main content of the object.

To add a caption to an object:

  1. Select the page containing the object in the Site Outline.
  2. Check the "Show Caption" box in the Wrap tab of the Inspector.
  3. Click the newly added caption text below the objects main content.
  4. Enter your caption.
  5. Click outside of the object to end editing.

Some page types (e.g. Photo or Video) also allow you to add a caption.

To add a caption to a suitable page:

  1. Select the page in the Site Outline.
  2. Check the "Show Caption" box in the Wrap tab of the Inspector.
  3. Click the newly added caption text below the objects main content.
  4. Enter your text.
  5. Click outside of the text to end editing.
