Site Map

A Site Map lists all the pages of a site. This is useful for more complex sites to aid visitors in finding the page they want.

The site map for a small site.

Note that Site Map pages are unrelated to Google Sitemaps that Sandvox Pro also offers integration with.

Settings

To edit a Site Map:

  1. Select it in the Site Outline.
  2. Open the Page Inspector. The top half contains general page attributes; the bottom holds settings specific to the Site Map as listed below.
