Site Map

A Site Map lists all the pages of a site. This is most useful for large sites, to aid visitors in finding the page they want.

The site map for a small site.

Note: Site Map objects are separate to Google Sitemaps.

Ways to add a Site Map to a site:

  • Use the "New" item in the toolbar.
  • Choose "New Page" from the Insert menu.

Site Map Options

To edit a Site Map:

  1. Select the site maps page in the Site Navigator.
  2. Click the site map image on the page to select it.
  3. Open the Object Inspector. Here you can edit settings specific to the Site Map as listed below.
