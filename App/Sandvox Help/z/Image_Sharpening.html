Image Sharpening

When creating thumbnail and favicon images, Sandvox automatically applies a sharpening effect to improve the image quality.

You can adjust how much of this effect is applied using Sandvox's hidden preferences. Please see the "Hidden Preferences" article for information on how to use these commands:

Favicon sharpening
defaults write com.karelia.Sandvox KTFaviconSharpeningFactor 1.0
Other image sharpening
defaults write com.karelia.Sandvox KTSharpeningFactor 0.3
