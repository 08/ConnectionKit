Toolbar

Like many modern Mac applications, Sandvox has a toolbar which allows access to many common functions.


You can customize the toolbar to better suit yourself.

Ways to customize the toolbar:

  • Choose "Customize Toolbar…" from the View menu.
  • Control-click the toolbar and choose "Customize Toolbar…" from the contextual menu.

A sheet will appear just below the toolbar containing all the toolbar items available in Sandvox.

Drag the items you require between the sheet and the toolbar. You can also drag the default set of items to return the toolbar to its original state.

The following toolbar items are available:

Pages
Adds a new page to the site. Equivalent to the "Site → Add Page" menu.
Pagelets
Adds a new pagelet to the page selected in the Site Outline. Equivalent to the "Site → Add Pagelet" menu.
Collections
Adds a new preset Collection to the site. Equivalent to the "Site → Add Collection" menu.
Group
Groups the pages selected in the Site Outline into a Collection. Equivalent to the "Site → Group as General Collection" menu.
Create/Edit Link
Creates or edits a link. Equivalent to the "Edit → Create/Edit Link…" menu.
Show/Hide Markers
Toggles Editing Markers on and off. Equivalent to the "View → Show/Hide Editing Markers" menu.
Designs
Shows or hides the Design Chooser. Equivalent to the "View → Show/Hide Designs" menu.
Setup Host
Opens the Host Setup Assistant. Equivalent to the "Site → Setup Host…" menu.
Publish Site
Publishes the site. Equivalent to the "Site → Publish Changes" menu.
Inspector
Shows or hides the Inspector. Equivalent to the "Window → Show/Hide Inspector" menu.
Media
Shows or hides the Media Browser. Equivalent to the "Window → Show/Hide Media Browser" menu.
Colors
Opens the standard Mac OS X color panel.
Fonts
Opens the standard Mac OS X font panel.
Separator, Space & Flexible Space
Used to space and separate items in the toolbar.
Customize
Opens the toolbar customization sheet.
