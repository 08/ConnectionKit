Sandvox Pro

Sandvox Pro allows access to the more advanced features of Sandvox. With Sandvox Pro you can:

  • Create Raw HTML pages and pagelets
  • Integrate your site with Google
  • Use Code Injection
  • View pages in the Web View:
    • Normally
    • Without styles
    • HTML source
    • DOM source
    • RSS feed
  • Validate pages
  • Paste HTML code into any text
  • Edit the raw HTML of any text area

To use Sandvox Pro, you download and install the same application onto your computer as for a regular copy. The only difference is the license key you enter into Sandvox which "unlocks" the Pro features.

If you have already paid for the regular edition of Sandvox, you can save money and only pay for the cost of upgrading to Sandvox Pro. Please see "Upgrading a Sandvox License" for more information. If not, you will have to pay the full price for Sandvox Pro.
