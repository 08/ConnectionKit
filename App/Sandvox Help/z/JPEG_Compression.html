JPEG Compression

With Sandvox you to specify the preferred format for images:

If you select JPEG, the quality (compression level) can be set for the JPEG images that Sandvox generates.

A high quality will result in better images, but larger file sizes (and therefore longer page loading times). A lower quality will give smaller files and quicker load times.

You can adjust the slider in the Document Inspector on the Appearance tab.
