Metrics Inspector

This article describes each of the settings available in the Metrics segment of the Inspector.


The Metrics segments is used to adjust Width and Height of an image to the pixel. You can make an image go back its original size by clicking the "Original Size" button.

You can check or uncheck the "constrain properties" box if you do not want the object to be height-width proportionate.
