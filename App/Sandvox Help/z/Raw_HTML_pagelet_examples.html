Raw HTML pagelet examples

Last.fm Pagelet

You can use a Raw HTML pagelet to make a last.fm pagelet.

To create a last.fm pagelet:

  1. Click the "Pagelets" item in the toolbar.
  2. Select "Raw HTML."
  3. Select the new Raw HTML pagelet by clicking on it.
  4. Open the Selection Inspector.
  5. Set the HTML content of the pagelet to: 
    <script src="http://imagegen.last.fm/chartjs/greybox/artists/ssp.js"></script>

Other Ideas

There are many websites that provide snippets of HTML that you can include into your own website. As long as the content they provide is no wider than 200 pixels wide, you should be able to add them to a raw HTML pagelet.

Some possible ideas:

  • From PayPal, a "buy now" or "donate" button
  • A Google Gadget such as a search form, video or maps.
  • Google Adwords
  • YouTube Video
  • Pandora Music List
  • Squidoo Lens
  • Weather.com conditions/forecast
