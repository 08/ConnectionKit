Embedded Page

An embedded page allows you to place the contents of another webpage inside a pagelet. This is sometimes also known as an iFrame. For example, a Google embedded page:

Pagelets are limited in width to 200 pixels, so it's a good idea to use a web page that is this width or narrower. If you do want to use a larger page, consider using an External Link set to "page within page."

Editing

To edit an embedded page, you use the Selection Inspector. In addition to those listed below, there are some general pagelet attributes that can be set.

Also, don't forget that you can click the small green Editing Marker () to insert some additional text into the pagelet.

URL

The address of the embedded page.

Height of page

The vertical space given to the page. If it is larger than this, a scroll bar will be added to enable navigation of the page. The width given to the embedded page is always 200 pixels.
