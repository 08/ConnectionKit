Callout

A callout is a pagelet set to appear within the main content of a page rather than the Sidebar

A pagelet set as a callout.

Any pagelet can be used as a callout, but some types of page do not allow callouts.

Callouts appear on the left or right of a page, it depends upon the design of the site.

To set a pagelet as a callout:

  1. Click the pagelet to select it.
  2. Open the Selection Inspector.
  3. Set the "Location" of the pagelet to "Callout."
