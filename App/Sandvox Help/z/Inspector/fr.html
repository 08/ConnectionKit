Sandvox possède un Inspecteur qui permet de configurer de nombreux réglages du site. Il existe six zones d'inspection :

  • Document
  • Page
  • Envelopper
  • Métrique
  • Lien
  • Objet

Les différentes manières d'afficher ou masquer l'inspecteur :

  • Cliquer sur l'élément "Inspecteur" dans la barre d'outils.
  • Choisir "Afficher l'Inspecteur" ou "Masquer l'Inspecteur" dans le menu Présentation.
Inspecteur de document (Raccourci : Commande-1)
Contient des réglages qui affectent la globalité du site
Inspecteur de page (Raccourci : Commande-2)
Contient des réglages qui affectent la page sélectionnée dans le canevas du site
Inspecteur pour envelopper (Raccourci : Commande-3)
Contient la commande Position de l'objet pour placer des objets sur la ligne, en légende, ou dans la barre latérale. Il existe des options de présentation, vous pouvez aussi spécifier une bordure pour vos objets (si le modèle l'autorise)
Inspecteur métrique (Raccourci : Commande-4)
Utilisé pour ajuster la largeur et la hauteur d'une image au pixel près
Inspecteur de lien (Raccourci : Commande-5)
Utiliser la commande Lien vers pour créer des liens vers des pages internes, des images en taille réelle, ou bien des URL externes
Inspecteur d'objet (Raccourci : Commande-6)
Utiliser ce segment pour configurer les réglages spécifiques à l'objet sélectionné
