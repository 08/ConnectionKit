Previous Step

The Host Summary

The Host Summary summarizes how your site is set up to be published. It contains the same information as the Host Overview.

Next Step

When you see the Host Summary, you have reached the end of the "Setting Up Your Host" procedure. You can either click the "Done" button to close the Host Setup Assistant and save your changes, or the "Go Back" button to make changes to your Host settings.
