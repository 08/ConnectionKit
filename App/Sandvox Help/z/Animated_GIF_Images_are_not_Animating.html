Animated GIF Images are not Animating

Sandvox is able to display GIF images, but it is unable to create them. If you add a GIF image (animated or not) to a site but it is too large to fit, Sandvox will resize it and convert the image to your preferred format (JPEG or PNG).

Therefore, to use an animated GIF within a site, you must ensure it is the correct size before inserting it into Sandvox. For example, embedded images must be no wider than 320 pixels, and Photo pagelets a maximum width of 200 pixels.

To stop Sandvox resizing the GIF image

  1. Remove the existing animated GIF from the site.
  2. Use a graphics editing program to resize the original GIF to the correct size.
  3. Insert the resized image into the site.

For more information on Sandvox's image resizing behavior, please see the "Images" article.
