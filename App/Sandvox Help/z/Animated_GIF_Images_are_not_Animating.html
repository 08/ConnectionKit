Animated GIF Images are not Animating

Sandvox fully supports animated GIFs. It will also allow you to convert a GIF image to PNG or JPEG format if desired. Note that if you do, the animation will be lost once published. If you've accidentally done this, remedy it by:

  1. Select the image
  2. Open the Object Inspector
  3. From the "Format" popup, select "GIF"
