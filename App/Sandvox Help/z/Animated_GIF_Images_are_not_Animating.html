Animated GIF Images are not Animating

Sandvox is able to display GIF images, but it is unable to create them. If you add a GIF image (animated or not) to a site but if it is too large to fit, it will need to be resized or it will no longer be animated.

Therefore, to use an animated GIF within a site, you must ensure it is the correct size.

To resize the GIF image

  1. Click on the GIF image in Sandvox.
  2. Open the Inspector to the Metrics Inspector.
  3. Resize the image to "Original Size"

For more information on Sandvox's image resizing behavior, please see the "Images" article.
