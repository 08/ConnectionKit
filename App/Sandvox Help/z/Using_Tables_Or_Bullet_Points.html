Using Tables Or Bullet Points

In Sandvox you cannot directly add tables or bullet points to a page. However, here is a handy workaround:

To insert a list:

  1. Open a rich text editor such as TextEdit
  2. Create a list
  3. Select the the list you have just created, and copy it to the clipboard
  4. Switch to Sandvox and paste the in the list

Alternately, you can use an HTML object.

To insert a table or bullet points using HTML:

  1. Add a raw HTML object from the Objects toolbar.
  2. In the "Edit Raw HTML window enter the appropriate HTML code, for example:"
<ul>
<li>Bullet 1</li>
<li>Bullet 2</li>
</ul>
