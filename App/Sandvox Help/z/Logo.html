Logo

Sandvox allows you to add a logo to your site; a small image displayed at the top of every page. Generally you use your company or organization's logo.

Karelia's logo on an example site.

To set a site's logo:

  1. Open the Site Inspector.
  2. Go to the Media tab.

  3. Drag an image to the "logo" box. This can be from the Media Browser, the Finder or another application.

To remove the Logo, click the cross in the bottom right of the box.

The maximum size of a logo in Sandvox is 200 x 128 pixels. If you try to use a larger image, Sandvox will resize it accordingly.

You should also set a description for the Logo. This is the text that will appear in place of the image for visitors using a web browser without image support. It is also useful for site access to the visually impaired.

Site Title

If the logo image you are using includes your site's title, you can use the Site Inspector to hide the site title to stop it appearing twice unnecessarily.
